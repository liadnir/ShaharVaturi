# Shahar's Carpentry Quote Generator

## Overview

This is an AI-powered conversational assistant designed to help the owner of a mobile carpentry workshop business quickly and accurately generate price quotes for clients.

The application guides the user through a simple, multi-step process to gather all the necessary information, calculates the costs and profit margins, and produces professional-looking outputs that can be sent directly to the client.

## Core Features

*   **Multi-Step Quote Generation**: A clean, step-by-step UI for entering client details, workshop parameters, and market-based profit adjustments.
*   **Internal Cost Breakdown**: A private, detailed breakdown of all costs (labor, travel, materials) and profit margins for the business owner's eyes only.
*   **Client-Facing Quote Summary**: A clear and concise text summary of the quote, ready to be copied or sent.
*   **PDF Export**: One-click generation of a professional PDF quote.
*   **Email Generation**: Pre-fills the user's default email client with a professionally written quote, ready to be sent to the client.
*   **Gemini AI Assistant**: An integrated chat interface powered by the Gemini API to help with writing client communications, brainstorming ideas, and answering business-related questions.

---

### **Important Note on Output Language**

To ensure maximum reliability and prevent rendering issues with custom fonts, all generated output files (**PDFs**) and pre-filled email drafts are formatted in **English**. The user interface remains in Hebrew for the business owner's convenience.

---

## Getting Started

This is a static web application with no backend. To run it, simply open the `index.html` file in a web browser. An internet connection is required for the application to load its dependencies and for the Gemini AI features to work.
